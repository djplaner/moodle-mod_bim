Generating an OPML file

Rather than use the Moodle/BIM interface to track when students make post, you can use the OPML file with a news reader. If you're a regular reader of blogs and other feeds, this can be a significant time saver.

You will still need to use Moodle/BIM to mark the contributions.

To do this you need to:

  1. Get a news reader.
  2. Download the OPML file.
  3. Import the OPML file into the reader.

News readers

If you don't already have a news reader, current recommendations are:

  • Mac user
    Download and install NetNewsWire
  • Web-browser reader
    Create an account and use Google Reader

Download the OPML file

Click on the link on "Your Students" screen.

Import your OPML file into your reader

How to do this will depend on your reader, but most will use the idea of "Importing Subscriptions"
