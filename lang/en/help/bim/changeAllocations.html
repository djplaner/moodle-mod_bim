Changing post allocations

Most BIM activities will have a specified list of questions, which students are expected to respond to by posting to their blog.

Before you can mark a student post, that post needs to be allocated to the question.

BIM attempts to automatically allocate student posts to specified questions by matching the content of the question title and the post title. But sometimes this does not work.

This page allows you to manually allocate posts that BIM has failed to automatically allocate.

It shows some details about the student and their blog and then a list of all the questions.

List of posts

All of the posts made by the student, that are known by BIM, are shown on this page.

For each post you should see a box that has the:

  • Status:; and

    A post can be: Unallocated, Submitted (Allocate, but not marked), Marked, Released and Suspended.

  • Title:.

    This is the title of the student post.

Within each box there will be:

  • Posted:;

    the time and date the student made the post public on their blog.

  • Change allocation to:

    This is the drop box you can use to change the allocation of the post. The drop box contains the titles of all the questions for which the student does not have a post. It also contains the option Unallocated which (not unsurprisingly) unallocates the post from a question.

    Whenever you select from this drop box, the form will automatically submit. Which means you can only change the allocation of one post at a time.

  • A big green box.

    The box should contain the full text of the student post.
