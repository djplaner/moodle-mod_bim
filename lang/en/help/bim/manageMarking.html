Manage marking

This page provides the coordinating teacher the ability to :

  • get an overview of and drill down into the detail of the marking by all teachers in a BIM activity; and
  • release marked assignments.

Overview

The overview of marking detail is provided by a table (e.g. view image of example table) that has a single row for every marker. For each marker the table will show:

  • Marker;

    The marker's name and a link to send them an email.

  • Studs;

    The number of students the marker has to mark.

  • A column for each question for this BIM activity.

Each question column for a marker will contain information about their marking progress for that question. This will take the form of 5 numbers, one number for each of the possible status settings (Submitted, Marked, Suspended, Released, Missing) for a post.

Drilling down

The example image may be helpful in understanding the following. From that image you may be able to see that the marker "Admin User" has 9 students who are "Missing" the post for the question "The Indicators Project". If you wish to drill down and find out more about these students (e.g. who are they), you click on the link that is "Missing".

Release marked posts

In the example image you may be able to pick out a number of links for "releasing" student posts. When you click on any of these links it will change the status of any Marked posts matching different criteria to released.

There are four different ways to release marked posts:

  1. Release all marked posts;

    In the example image you may be able to see a link Release all 3 marked post(s). Clicking on this link will release all of the marked posts (only 3 in this example).

  2. Release all marked posts for a given question;

    In the example iamge you may be able to see the links release under the question titles The indicators project and Moodle. Clicking on these links will release all marked posts for that question.

  3. Release all marked posts for a given marker;

    In the example image you may be able to see the links release under the markers' names Marilyn Wells, David Hamilton and Kevin Bell. Clicking on these links will release all posts marked by the particular marker.

  4. Release all marked posts for a question and a marker;

    In the example image you may be able to see release links in a number of cells for specific markers/questions. For example, in the cell for The Indicators Project and Marilyn Wells. Clicking on these links will return only those posts marked by that marker for that question.

Note: Only those posts in the Marked state are returned.
